Question Bank — jQuery & AngularJS – Answers (Concise)

Answers are concise, follow the suggested key points and include required code snippets/diagrams labels where indicated.

Section A – 5 Marks Questions (20)

1. Define jQuery. Why is it called "write less, do more"?

Answer: jQuery is a lightweight JavaScript library that simplifies HTML DOM traversal/manipulation, event handling, animation and AJAX. "Write less, do more" means common tasks require fewer lines of code than plain JavaScript due to concise selectors and utility methods.

2. Syntax to include jQuery in a webpage

<!-- CDN -->
<script src="https://code.jquery.com/jquery-3.6.0.min.js"></script>

<!-- Local copy -->
<script src="js/jquery.min.js"></script>

3. Differentiate between id and class selectors in jQuery

Answer: $("#id") selects a single element by its id (unique). $(".class") selects one or more elements with that class. Example: $("#header").hide(); $(".item").addClass("active");

4. Role of ready() in jQuery

Answer: $(document).ready(function(){ ... }); ensures the DOM is fully loaded before running code so elements can be accessed reliably.

5. jQuery snippet to hide a paragraph on button click

<button id="b">Hide</button>
<p id="p">Some text</p>
<script>
  $("#b").click(function(){
    $("#p").hide();
  });
</script>

6. Define AngularJS

Answer: AngularJS (1.x) is a client-side MVC/MVVM JavaScript framework by Google providing two-way data binding, dependency injection, directives and services to build single-page applications.

7. What is ng-model in AngularJS?

Answer: ng-model binds form inputs (input, textarea, select) to properties on the scope, enabling two-way data binding between view and model.

8. Differentiate between ng-show and ng-hide

Answer: ng-show="expr" sets CSS display based on truthiness (shown when true). ng-hide="expr" hides when expression is true (inverse). Both toggle visibility without removing element from DOM.

9. Short note on AngularJS expressions

Answer: Expressions ({{ ... }}) are evaluated against the current scope to render values. They are not full JavaScript (no control flow) and are used for simple calculations and string concatenation.

10. What is $scope in AngularJS?

Answer: $scope is the object that links controllers and views; it stores model data and functions accessible in the template.

11. Any two advantages of jQuery over JavaScript

  1. Cross-browser compatibility abstraction.
  2. Concise syntax for DOM selection/manipulation and built-in effects/AJAX helpers.

12. Syntax for chaining multiple filters in AngularJS

Answer: {{ data | filter1 | filter2 }} for sequential filter application.

13. List any three jQuery event methods

Answer: click(), hover(), keydown(). (Also on(), submit()).

14. What is ng-repeat?

Answer: ng-repeat="item in items" repeats a DOM template for each item in a collection, creating child scopes for each iteration.

15. Differentiate AngularJS directive and expression

Answer: Directives are markers (attributes/elements/classes) that extend HTML behavior (ng-model, ng-repeat). Expressions evaluate and display values inside templates ({{ }}).

16. Difference between text() and html() in jQuery

Answer: .text() gets/sets text content (escaped). .html() gets/sets HTML content and can insert markup.

17. Explain AngularJS filters briefly

Answer: Filters format data for display (e.g., currency, uppercase, orderBy, custom filters). Use in templates with | pipe.

18. Syntax to create a custom directive in AngularJS

<script>
angular.module('app',[])
  .directive('myDirective', function(){
    return {
      restrict: 'E',
      template: '<div>Hello</div>'
    };
  });
</script>

19. Define AJAX in jQuery

Answer: AJAX in jQuery refers to asynchronous HTTP requests using helpers like $.ajax(), $.get(), $.post() to fetch/send data without reloading the page.

20. Any two AngularJS form validation states

Answer: $dirty / $pristine, and $valid / $invalid.

Section B – 15 Marks Questions (20)

1. Explain jQuery DOM traversal methods with examples

Answer: parent() selects immediate parent; children() selects direct children; find(selector) searches descendants; siblings() selects sibling elements. Example:

$('#list').children();
$('#item').parent().addClass('p');
$('#container').find('p.highlight');
$('.active').siblings().removeClass('active');

2. jQuery effects: fadeIn(), fadeOut(), slideUp(), slideDown()

Answer: Basic animations to change visibility over time. 

$('#box').fadeIn(400);   // show with fade
$('#box').fadeOut(400);  // hide with fade
$('#panel').slideUp(300); // hide by sliding up
$('#panel').slideDown(300); // show by sliding down
// Can pass callback: $('#box').fadeIn(400, function(){ /* after complete */ });

3. Compare JavaScript, jQuery, and AngularJS

Answer:

  • JavaScript: Language providing DOM API, events, XHR; imperative coding.
  • jQuery: Library built on JS simplifying DOM selection/manipulation, events, animations, AJAX; good for progressive enhancement.
  • AngularJS: Full MVC/MVVM framework for building SPAs with two-way binding, dependency injection, directives; more opinionated and declarative.

4. Explain AngularJS two-way data binding with an example

Answer: Changing model updates view and user input updates model automatically via ng-model. Example: 

<input ng-model="name" />
<p>Hello {{name}}</p>

5. jQuery program to toggle visibility of a paragraph

<button id="t">Toggle</button>
<p id="p">Text</p>
<script>
  $('#t').click(function(){
    $('#p').toggle();
  });
</script>

6. Explain AngularJS directives with any 3 examples

Answer:

  • ng-model: binds input to scope property.
  • ng-repeat: repeats DOM for collections.
  • ng-bind: binds expression to element text to avoid flicker.

7. Differentiate between AngularJS controller and service

Answer: Controller: attaches data/functions to $scope for a view. Service: singleton reusable component for business logic or data access injected via DI. Controllers are for view logic; services for reusable logic.

8. Explain jQuery AJAX methods with example

Answer: $.ajax() for full control, $.get()/$.post() shortcuts. Example:

$.getJSON('data.json', function(data){
  // process data
});

// or
$.ajax({ url: '/api/items', method: 'GET', success: function(resp){ console.log(resp); } });

9. What are AngularJS modules? Steps to create and use a module

Answer: Modules are containers for app parts (controllers, services, directives). Create with angular.module('app',[]). Attach controller: angular.module('app').controller('MainCtrl', function($scope){...});. Add ng-app="app" to HTML root.

10. Explain AngularJS form validation with example

Answer: Use ng-model and HTML validation attributes like required, ng-minlength. Example: 

<form name="f" novalidate>
  <input name="email" ng-model="user.email" required type="email" />
  <div ng-show="f.email.$touched && f.email.$invalid">Invalid email</div>
</form>

11. Explain event delegation in jQuery. Why needed?

Answer: Event delegation attaches handler to a parent and specifies selector, e.g. $(parent).on('click', '.child', handler). Needed for dynamic elements added after binding and for efficiency (fewer handlers).

12. Differentiate ng-bind and expressions ({{ }})

Answer: ng-bind="expr" sets element text without initial flicker (pre-binding). {{expr}} is inline and may momentarily show raw template before Angular compiles.

13. Short note on AngularJS scope hierarchy

Answer: Scopes form a prototypal hierarchy: $rootScope at top; each directive/controller may create child scopes inheriting prototypally. Changes propagate following scope resolution rules.

14. Role of $http service in AngularJS with example

Answer: $http performs AJAX calls returning promises. Example:

app.controller('C', function($scope, $http){
  $http.get('/api/data').then(function(resp){ $scope.items = resp.data; });
});

15. jQuery code to validate a simple login form

$('#login').submit(function(e){
  e.preventDefault();
  var u = $('#user').val(), p = $('#pass').val();
  if(!u){ alert('Enter username'); return; }
  if(p.length < 6){ alert('Password too short'); return; }
  // submit via AJAX or form.submit()
});

16. Explain AngularJS filters with example of custom filter

app.filter('reverse', function(){
  return function(input){ return (input||'').split('').reverse().join(''); };
});

// usage: {{ 'abc' | reverse }} -> 'cba'

17. jQuery script to change CSS properties dynamically

$('.box').css({ 'font-size': '18px', 'background-color': '#f0f0f0' });
// or
$('#id').css('color','red');

18. MVC architecture of AngularJS

Answer: Model = data on scope; View = HTML templates; Controller = sets up scope. Angular's directives bind view and controller; $scope acts as bridge. The digest cycle keeps model & view in sync.

19. Explain AngularJS $watch with example

$scope.$watch('value', function(newVal, oldVal){
  if(newVal !== oldVal) { /* react to change */ }
});
// Used to run code when a scope expression changes.

20. Difference between AngularJS 1.x and jQuery re: data binding

Answer: AngularJS is declarative and data-driven with two-way binding and templating; jQuery is imperative DOM-manipulation focused where the developer updates DOM manually.

Section C – 20 Marks Questions (10)

1. Design a small jQuery-based image gallery with next/prev buttons

Answer (outline & code):

<div id="gallery">
  <img id="gimg" src="img1.jpg"/>
  <button id="prev">Prev</button>
  <button id="next">Next</button>
</div>
<script>
var imgs=['img1.jpg','img2.jpg','img3.jpg'];
var i=0;
$('#next').click(function(){ i=(i+1)%imgs.length; $('#gimg').attr('src', imgs[i]); });
$('#prev').click(function(){ i=(i-1+imgs.length)%imgs.length; $('#gimg').attr('src', imgs[i]); });
</script>

Explain: array of image paths, index update on click, update src. Use fadeIn/fadeOut for transition.

2. AngularJS application for student result management (ng-repeat & filters)

Answer (structure & sample):

<div ng-app="app" ng-controller="C">
  <input ng-model="q" placeholder="search"/>
  <table>
    <tr ng-repeat="s in students | filter:q | orderBy:'name'">
      <td>{{s.name}}</td>
      <td>{{s.marks}}</td>
    </tr>
  </table>
</div>

app.controller('C', function($scope){
  $scope.students = [{name:'A',marks:85},{name:'B',marks:72}];
});

Explain: ng-repeat iterates, filters search/order, can add custom filters for grade calculation.

3. Explain AngularJS form validation in detail with code

<form name="regForm" novalidate>
  <input name="email" ng-model="user.email" required type="email"/>
  <span ng-show="regForm.email.$touched && regForm.email.$invalid">Invalid email</span>
  <input name="pass" ng-model="user.pass" required ng-minlength="6"/>
  <span ng-show="regForm.pass.$error.minlength">Too short</span>
  <button ng-disabled="regForm.$invalid">Submit</button>
</form>

Explain: built-in states ($dirty, $pristine, $valid, $invalid), using $error to show messages, custom validators via $validators or directives.

4. jQuery registration form validation (name,email,password,confirm)

$('#reg').submit(function(e){
  e.preventDefault();
  var name=$('#name').val(), email=$('#email').val(), p=$('#pass').val(), cp=$('#cpass').val();
  var emailRe=/^[^\s@]+@[^\s@]+\.[^\s@]+$/;
  if(!name){ alert('Name required'); return; }
  if(!emailRe.test(email)){ alert('Invalid email'); return; }
  if(p.length<6){ alert('Password min 6'); return; }
  if(p!==cp){ alert('Passwords do not match'); return; }
  // submit
});

5. Explain AngularJS dependency injection with example

Answer: DI injects services by name into components. Example: app.controller('C', function($scope, $http){ ... });. This makes components testable and modular; services are singletons provided by Angular.

6. Compare AngularJS Directives and Components

Answer: Directives extend DOM (attribute/element). Components (Angular 1.5+) are a special directive configuration for building reusable UI with isolated scope and lifecycle hooks. Components encourage one-way bindings and are easier to migrate to newer Angular.

7. Create jQuery + AJAX example to fetch and display data from JSON file

$.getJSON('students.json', function(data){
  data.forEach(function(s){
    $('#list').append('<li>'+s.name+' - '+s.marks+'</li>');
  });
});

8. Discuss the digest cycle in AngularJS (diagram label recommended)

Answer: The digest cycle checks watched expressions for changes and calls corresponding listeners. Steps: 1) Model change or event triggers $apply → 2) $digest runs → 3) Iterate watchers until stabilization (dirty-check loop) → 4) Update view. Diagram: $scope change → $apply() → $digest() → watch functions → listener functions → DOM update.

9. Build a small AngularJS SPA with routing (outline)

// include ngRoute, configure routes
angular.module('app',['ngRoute'])
.config(function($routeProvider){
  $routeProvider
   .when('/home',{templateUrl:'home.html',controller:'HomeCtrl'})
   .when('/list',{templateUrl:'list.html',controller:'ListCtrl'})
   .otherwise({redirectTo:'/home'});
});

Explain: ng-view placeholder, controllers per route, partials as views.

10. Differences between jQuery and AngularJS re architecture and data binding

Answer: jQuery manipulates DOM directly; AngularJS provides MVC/MVVM structure with declarative templates and automatic binding. Use jQuery for small DOM tasks; AngularJS for full app architecture and data-driven UIs.
